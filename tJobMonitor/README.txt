Welcome to Talend tJobMonitor component
=======================================================================

This component support that administrator will be known information of job and system resources by various option, measured value, the numbers. 
You can setting various unit and selecting resource options. 
This able to setting periodic probe point, measured data unit, and memory & CPU resource of job and system.
